A modern web app store built with React, Tailwind, and DaisyUI.

## Project Overview

HeroApp is a responsive web application that showcases mobile applications, allowing users to explore, search, and install apps directly. The platform includes home, all apps, app details, and "My Installation" pages with smooth navigation and interactive UI components.

This project is built following modern web development standards and deployed online for easy access.
